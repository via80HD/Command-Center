Brave repository error when trying to run ‘sudo apt update’ that says’ “…doesn’t support architecture ‘i386’ error”

1. Go to the sources directory:

cd /etc/apt/sources.list.d/

2. List the files:

ls

3. Open the source list file with nano:

sudo nano /etc/apt/sources.list.d/

4. On the screen you are taken to, you will need to click the right arrow 5 times then type the following

arch=amd64

5. Save and exit with Ctrl+X, then type ‘y’ & press Enter.

6. Update package list:

sudo apt update
